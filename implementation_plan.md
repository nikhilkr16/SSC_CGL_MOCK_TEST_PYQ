## Proposed Changes

### Phase 1 — PDF Parser & Question Database

#### [NEW] [parse-pyq.js](file:///e:/learn/SSC%20CGL/mock-test-app/scripts/parse-pyq.js)
Node.js script using `pdfjs-dist` to:
- Parse all 84 PDFs from `PYQ/`
- Extract question text, options A/B/C/D
- Detect green-colored text → mark as `correct_option`
- Detect section headers → tag `QUANT | REASONING | GS | ENGLISH`
- Auto-tag topics via keyword matching
- Parse explanation/answer-key sections at end of PDFs
- Output `data/questions.json`
